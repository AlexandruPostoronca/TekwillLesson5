Meteorological seasons

Rewrite the following switch statement as a switch expression using the code template below. 


int temperature = 0;
switch (season) {
    case SPRING:
    case AUTUMN:
        temperature = 20;
        break;
    case SUMMER:
        temperature = 37;
        break;
    case WINTER:
        temperature = 1;
        break;
    default:
        throw new IllegalStateException("Invalid name of Seasons.");
}
